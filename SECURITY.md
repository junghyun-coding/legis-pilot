# 보안 정책 (Security Policy)

법령이음(Legis-Pilot)의 보안 취약점 보고 및 비밀키 관리 정책입니다.

## 지원 범위

보안 패치는 기본 브랜치(`master`)의 최신 코드에만 적용됩니다. 과거 커밋·태그·포크본은
지원하지 않습니다.

## API 키 관리

- OpenAI 및 국가법령정보 Open API 등 모든 비밀키는 `.env.local`로만 관리합니다.
- `.env.local`은 절대 커밋하지 않습니다(`.gitignore`로 차단된 상태를 유지하세요).
- 키를 소스 코드, 커밋 메시지, 이슈, PR, 스크린샷, 로그 어디에도 노출하지 마세요.
- 키가 노출되었거나 노출이 의심되면 **즉시 해당 키를 폐기(revoke)하고 재발급**하세요.
